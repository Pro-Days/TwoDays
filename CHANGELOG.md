# CHANGELOG

이 문서는 프로젝트의 `업데이트 내역`과 `업데이트 예정`을 관리합니다.

- 형식: 사람이 읽기 쉬운 요약 중심
- 원칙: 확정된 변경은 `업데이트 내역`, 계획/검토 중인 항목은 `업데이트 예정`에 기록
- 주의: `업데이트 예정` 항목은 우선순위/일정이 변경될 수 있음

## 업데이트 예정

아래는 향후 반영 후보/계획을 기록하는 영역입니다.

#### 기능 개선 (검토중)

- 캐릭터 정보 비교 기능 추가 (레벨, 랭킹 비교)
- `/랭킹` 히스토리 가시성 향상 및 특정 캐릭터 강조 옵션 추가
- `/검색`에서 여러 캐릭터를 한 번에 표시하는 기능 추가

#### 성능/호출 최적화 (검토중)

- `uuid`, `name` 조회 횟수 감소 (코드 흐름 개선 / 캐시 활용)

#### 내부 구조/코드 품질 개선 (검토중)

- 프로그램 전역 상수 정리 (예: `HTTP_TIMEOUT`)

#### 사용자 경험 개선 (검토중)

- 유저가 봇 응답 메시지를 삭제할 수 있는 기능 추가

### 신규 명령어 아이디어 (검토중)

#### `/정보` 명령어 아이디어

- `npc`: 모든 NPC 좌표 + 이동 방법 + 관련 퀘스트 정보 조회 (콤보박스 형태 검토)
- `퀘스트`: 모든 퀘스트 정보 조회 (콤보박스 형태 검토)
- `질문`: 자주 묻는 질문(FAQ) 또는 팁 조회 (콤보박스 형태 검토, /검색으로 대체 가능성 검토)

#### `/계산` 명령어 아이디어

- `레벨업`: 사냥 + 퀘스트 기준 최적 레벨업 루트 계산

#### `/검색` 명령어 아이디어

- 자주 묻는 질문(FAQ)를 검색하는 옵션 추가 (AI 또는 사전 정의된 FAQ 데이터 활용 검토: 시맨틱 캐싱)

#### `/질문` 명령어 아이디어

- 봇에게 질문을 하면 답변을 받기로 설정한 유저들에게 질문을 보내고 답변을 받는 기능
